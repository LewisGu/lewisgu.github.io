Vivado安装教程补丁
FPGA Vivado
IDE配置
Publish Date:   2017-08-05
Word Count:   361
Read Times:   1 Min
Read Count:  

最近开始搞FPGA，用的Xilinx公司的板子，写Verilog，自然也就得用Xilinx自家的软件Vivado，版本2015.4。

安装配置网上也挺多教程的，选了个CSDN上的参考了下 VIVADO 安装教程这个教程。

在添加license后这步，我使用的是vivado长期有效license这个，确保不会因为license问题导致后续编译出错。

然后下载vivado.2015.4.1.win64 license这个资源，解压后将vivado文件夹下lib文件夹中的文件复制到安装路径下，重启软件。

如果是2016.1之后版本的Vivado，可以参考下【入门必看】学习Vivado如何获取License这个指南，并没有使用过，无法验证其可靠性。

分享两个代码段，分别是《Xilinx FPGA设计权威指南 Vivado集成设计环境》一书中第二章的设计代码以及测试代码，经验证在2015.4版本的vivado上测试通过，可以来验证自己软件的安装配置是否正确。

top.v 设计代码

module top(
    input a,
    input b,
    output[5:0] z
    );
assign z[0] = a&b;
assign z[1] = ~(a&b);
assign z[2] = a|b;
assign z[3] = ~(a|b);
assign z[4] = a^b;
assign z[5] = a~^b;
endmodule

endmodule

test.v 测试代码

module test;
reg a;
reg b;
wire[5:0] z;
top uut(
    .a(a),
    .b(b),
    .z(z)
);

initial

begin
    while(1)
        begin
            a=0;b=0;#100;
            a=0;b=1;#100;
            a=1;b=0;#100;
            a=1;b=1;#100;
        end
    end

endmodule
